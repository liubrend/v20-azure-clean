# Architecture Decision Records

One file per decision, numbered `0001-*.md`, `0002-*.md`, … Records are immutable:
to change a decision, add a new ADR that supersedes the old one (don't rewrite it).

Copy `adr-template.md` to start. Keep titles short and outcome-oriented
(e.g. `0001-postgresql-as-system-of-record.md`).
